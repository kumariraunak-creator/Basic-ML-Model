# Basic Machine Learning Model

## Overview
This project implements a Logistic Regression model on the Titanic dataset using Scikit-learn.

## Dataset
Titanic Dataset

## Libraries Used
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## Workflow
1. Load Dataset
2. Data Exploration
3. Train-Test Split
4. Train Logistic Regression Model
5. Make Predictions
6. Evaluate Accuracy
7. Generate Classification Report
8. Plot Confusion Matrix

## Results
Accuracy: 73.18%
